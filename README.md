# Maria-Manuela Bîrnat — React Portfolio

A responsive portfolio built during **Technigo's Web Development Boot Camp (2024)**, based on the official [Figma design](https://www.figma.com/design/ZMIWBimuoSDHXp7CdtkrA3/Web-Dev-Portfolio-%7C-Fall-2024?node-id=1862-60).

## Live site

[https://bmmportfolio.netlify.app](https://bmmportfolio.netlify.app)

## About the project

This was my **project week 8** assignment: a portfolio site that pulls featured GitHub repos via the API, matches them with local project data, and presents them with reusable React components.

The layout is split into focused sections — intro, tech, projects, articles, skills, and contact — with shared UI components in a common folder.

## Tech stack

- React + Vite
- JavaScript (JSX)
- CSS (responsive layout, Figma breakpoints)
- GitHub REST API
- Netlify (deployment)

## Updates

The project was originally built in **2024** and has since been refined — especially responsiveness, project fetching, and layout alignment across breakpoints.

Parts of the recent work were done with **[Cursor](https://cursor.com)** (AI-assisted coding). I used it as a pair-programming tool: reviewing suggestions, testing in the browser, and keeping changes that matched the Figma specs and my own code style. The architecture, content, and final decisions remain mine.

## Getting started

```bash
cd my-react-portfolio
npm install
npm run dev
```

The app runs at `http://localhost:5173`.
